# Birthday Beat Showcase

A special interactive web application for showcasing a birthday EP.

## Features

- Custom audio player for multiple tracks
- Responsive design for mobile and desktop
- Interactive UI with glassmorphism effects
- Automatic deployment to GitHub Pages

## Tech Stack

- React (TypeScript)
- Vite
- Tailwind CSS
- Lucide Icons
- Radix UI

## Local Development

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Build for production: `npm run build`

## Deployment

This project is configured to automatically deploy to GitHub Pages via GitHub Actions when changes are pushed to the `main` branch.
